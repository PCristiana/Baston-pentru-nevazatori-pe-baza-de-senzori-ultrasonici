Baston Inteligent pentru Nevăzători
Acest proiect reprezintă o soluție accesibilă și practică pentru persoanele cu deficiențe de vedere, oferindu-le un instrument care să le ajute în detectarea obstacolelor din mediul înconjurător. Cu ajutorul unui senzor ultrasonic HC-SR04, bastonul măsoară distanța până la obiecte și reacționează diferit în funcție de apropiere.

Pentru a transmite informația în mod intuitiv, utilizatorul este avertizat prin vibrații (generate de un motor vibrator de 3V) și sunete (redate de un buzzer piezoelectric). Intensitatea vibrației și complexitatea melodiei cresc pe măsură ce obstacolul se apropie, oferind un feedback clar și eficient.

Controlul tuturor componentelor este realizat de o placă Arduino Uno R3, programată să gestioneze logica de funcționare. Proiectul este alimentat de patru baterii AA (6V), iar componentele sunt fixate pe o placă PCB, pentru un montaj sigur și portabil.

Prin simplitatea și eficiența sa, acest baston inteligent poate fi o bază solidă pentru extinderi viitoare – cum ar fi integrarea GPS, Bluetooth sau aplicații mobile – și contribuie la crearea unui mediu mai sigur și mai independent pentru persoanele nevăzătoare.
